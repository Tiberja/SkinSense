# SkinSense - Goals

## Gruppenziele
Wir planen ein solides, gut funktionierendes MVP, das alle Kernfunktionen enthält:
- Account/Login-System
- Produktfinder + Filter
- Produktdatenbank (15–20 Beispielprodukte)
- Favoriten-System
- Sauberes UI + klare Navigation
- Dokumentation & Versionsverwaltung (GitHub)

## Persönliche Ziele :
**Acelya** 
Backend mit Python + Flask aufsetzen
API-Endpunkte für:
- Hauttyp-Finder
- Produktsuche
- Filter
Verbindung mit einer SQLite-Datenbank
- Favoriten/Produktdetails anzeigen
- UI-Design dokumentieren

**Tiberja**
